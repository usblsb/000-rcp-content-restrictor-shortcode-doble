Ayuda para el Plugin: Enclosed Content Restrictor

¿Qué hace este plugin?

Este plugin permite restringir el contenido encerrado en el shortcode basado en roles de usuario, capacidades específicas y niveles de membresía. Es especialmente útil para sitios que ofrecen contenido premium o exclusivo para ciertos usuarios.

¿Cómo funciona?

El plugin utiliza el shortcode [enclosed_content_restrictor] ... [/enclosed_content_restrictor] para envolver el contenido que deseas restringir. Puedes especificar el rol, la capacidad y los niveles de membresía necesarios para acceder a este contenido.

Utilidad del Plugin

Ideal para sitios de membresía, educativos o de contenido premium, este plugin asegura que solo los usuarios con los privilegios adecuados puedan acceder a contenido específico.

¿Cómo se usa?

Aquí tienes algunos ejemplos de cómo puedes utilizar el shortcode:

  • Restringir contenido solo a usuarios con un rol específico:
    [enclosed_content_restrictor role="rol_deseado"]...[/enclosed_content_restrictor]
  • Restringir contenido basado en una capacidad específica del usuario:
    [enclosed_content_restrictor capability="capacidad_deseada"]...[/enclosed_content_restrictor]
  • Restringir contenido a niveles de membresía específicos:
    [enclosed_content_restrictor membership_levels="1,2"]...[/enclosed_content_restrictor]

Puedes combinar estos atributos para crear reglas de acceso más específicas. Por ejemplo, para restringir contenido a usuarios con un rol específico y un nivel de membresía, puedes usar:
[enclosed_content_restrictor role="rol_deseado" membership_levels="1,2"]...[/enclosed_content_restrictor]

Nota: Si desactivas el plugin, el contenido encerrado entre los shortcodes será visible para todos los usuarios.
